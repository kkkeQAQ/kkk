# playball (with kkke toolkit)

play ball in shell terminal

a example of ktk

## Dependencies

### Build dependencies

* libcurses
* libpthread
* c++11
* cmake >= v2.6


## Installation

### Build from source code

1. Make sure you have installed all dependencies.

2. Build:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

3. Install:
```
$ sudo make install
```
